# 基于机器学习的电机故障在线监测

## Abstract

主要思路来自于《基于机器视觉的电机故障识别方法及实验研究》。首先通过Maxwell，对某型号异步电机进行建模及仿真，获取正常情况、短路情况以及断路情况下的电流、转矩波形。然后建立了两类机器学习模型：SVM和随机森林，SVM通过识别电流Park矢量图来判断故障情况，随机森林由于是非监督模型因此只能判断是否处于故障。使用pyqt设计了主要交互界面。具体实现功能请参考`总结.pptx`
